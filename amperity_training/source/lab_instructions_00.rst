==================================================
Pre-lab: Check access
==================================================

In this class, you will be completing exercises that are tailored specifically for your training exercises. Prior to the start of class, please follow the steps below to check that you have access to your student tenant.

Verify access to lab
==================================================

#. Using Google Chrome, navigate to https://app.amperity.com/

   .. note:: If you do not already have Chrome, please install it. Amperity requires Google Chrome (or a Chromium-based browser).

#. Log in to Amperity. If you've never before logged into Amperity, log in using your email address and reset your password. If you normally log into Amperity using single sign-on (SSO), a password will not be required.

#. In the top-right corner, select the tenant name and select **View all tenants**.

   .. image:: /_images/ex00_tenant_selection_menu.png
      :width: 300 px
      :alt: Tenant selection menu
      :align: left
      :class: no-scaled-link

#. From the **All available tenants** modal, select the training tenant you have been assigned below. Your training tenant name should start with "TET" and contain your name.

   .. image:: /_images/ex01_available_tenants_modal.png
      :width: 500 px
      :alt: Available tenants modal
      :align: left
      :class: no-scaled-link

#. The training tenant name should be displayed in the top-right corner.

   .. image:: /_images/ex01_tenant_name_display.png
      :width: 300 px
      :alt: Tenant name display
      :align: left
      :class: no-scaled-link

#. Confirm that you have access to ALL 9 pages along the left-hand navigation: **Sources**, **Stitch**, **Customer 360**, **Queries**, **Segments**, **Activations**, **Destinations**, **Workflows**, and **Settings**.

.. note:: All sample files used in these labs are available here. This includes the files that will be ingested in the course labs and files that were pre-ingested on your student tenant, so you are free to experiment with them on your own.
