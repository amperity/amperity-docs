.. _sendto-cheetah-digital-build-query:

Build query
==================================================

.. sendto-cheetah-digital-build-query-start

Build a query that returns the customer profile data that you want to send to |destination-name|. For example:

.. code-block:: sql

   SELECT
     email AS "email"
     ,phone AS "phone"
     ,address AS "address"
     ,city AS "city"
     ,state AS "state"
     ,postal AS "zip"
   FROM Customer360
