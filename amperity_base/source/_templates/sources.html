Sources

A source is any external system from which Amperity can pull customer profile and interaction records.

Adobe Analytics

Pull clickstream data from Adobe Analytics using SFTP.

Adobe Campaign

Pull data from Adobe Campaign using SFTP.

Adobe Commerce

Adobe Customer Attributes

Pull data from Adobe Customer Attributes using SFTP.

Amazon S3

Attentive

Pull customer SMS data from Attentive using SFTP.

Azure Blob Storage

Bazaarvoice

Pull data from Bazaarvoice using SFTP.

Bluecore

Pull audience exports from Bluecore using SFTP.

Braze

Dynamic Yield
